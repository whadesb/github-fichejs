Les variables :

Les variables sont des boites qui stockent des données. Elles peuvent etre de type "String" et contenir du texte (nom d'utilisateur, textes...)! Elles peuvent contenir des données, c'est à dire des nombres.

Exemple d'une varible de type texte : let nomDeLaVariable = "texte de la variable";
Exemple d'une variable de type nombre : let nomDeLaVariable = 18;
